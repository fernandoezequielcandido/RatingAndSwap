Welcome to Laavor Rating and Swap Image/Label!





Rating Image - With a unique class you can creating a rating, with a event using a property 'commandEvent'.





InitialValue inform the selected value when starting without user click. You can set ItemsNumber telling the rating size (number of images). Property imageDeselect is preview click image and imageSselect is the post click image.




You can change width and height of images using imageWidth/imageHeight, if blockSelect is 'true' the user can change rating only once.





Instead of images use the numerical rating. It is possible to change as text, background and border kernels easily. With spaceBetween you can define the distance between values, also available in Rating Image.





An interesting new rating for letters, giving a different usability to the user.





Easily swap images with only one html tag, indicating the first and second images, with a simple event and you can control the functionality you need.




Easily change text with one touch and change color and background to enhance user usability.
